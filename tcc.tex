\begin{resumo}{OLAP, InhA, Docagem Molecular}

O processo de desenvolvimento de novos fármacos é uma atividade fundamental para a indústria farmacêutica, na qual é utilizado não somente para o desenvolvimento de novos compostos para atender uma determinada necessidade, mas também tem como objetivo evoluir os compostos já existentes em busca de um resultado aprimorado e mais eficaz. 

A área de bioinformática tem como um de seus objetivos, através do uso de recursos e técnicas computacionais, acelerar e aprimorar o processo de desenvolvimento de medicamentos. Esse processo consiste na análise das interações entre pequenas moléculas e receptores, e o processo de docagem molecular é o responsável por simular e avaliar essas interções.  

Porém, a quantidade de dados gerados por estes experimentos cresce em função do conjunto de snapshots que é utilizado, dificultando a análise e visualização de todos os resultados, que é feito de forma manual. Com base nisso, foi desenvolvido um modelo de dados OLAP que permite ao especialista de domínio uma análise multidimensional dos resultados de experimentos de uma forma mais ágil e precisa. O uso desta solução permite trabalhar com as informações de uma maneira flexível, podendo servir para identificar comportamentos baseados em dados históricos, realizar consultas complexas e também obter relatórios baseados em dados históricos, sem necessitar de conhecimentos técnicos e de recursos computacionais de alta capacidade.

\end{resumo}

\begin{abstract}{OLAP, InhA, Molecular Docking}

The drug development process is an very important activity for pharmaceutical industry, not just to develop new drugs but also to improve compounds who already are used, making them more effective. 

The bioinformatics field have as one of their objectives, through the use of computational resources, speed up and improve the process of drug development. This process consists in the analysis of interactions between small molecules and receptors, and the molecular docking process is the responsible to simulate and evaluate these interactions. 

However, the amount of data generated by these experiments increases as the size of snapshot set used making difficult the visualization and analysis of all resultant data, which is done manually. Based on this, an OLAP model was developed for allow specialists to perform an multidimentional analisys in resultant set of experiments in a quickly and precise way. The use of this solution allow to manage data in a flexible way, allowing to identify behaviors based in historical data, make complex queries and generate data reports based on historical data without need technical knowledge and high capacity computers.

\end{abstract}
